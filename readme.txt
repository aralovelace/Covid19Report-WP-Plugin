== Description ==

Simple Covid19APIReport is a plugin that let you add covid19 report WordPress site.


== Requirements ==

*   This plugin is only compatible with PHP version 7.0 and above.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/Covid19APIReport` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Then you can now add the shortcode  <code>[covid19-reports]</code>  in the Post, Page or HTML Widget.
